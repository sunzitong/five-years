人脸信息使用授权协议

1. 如您同意本协议，则表明您自愿授权冠寓运营方使用人脸识别技术采集、处理您本人的个人信息，用于识别您本人身份；
2. 如您不同意本协议内容，请勿录入您的人脸信息数据，与此对应，您将无法通过人脸识别方式进出冠寓门店；尽管如此，您仍可通过线下身份核对、门禁卡感应刷卡、冠寓APP扫码等其他方式进入冠寓门店；
3. 使用过程中，您随时可以通过冠寓APP自行删除已录入的人脸信息数据；
（注：所谓数据删除并非物理删除，在您完成删除操作后，系统会在后台将您的人脸信息数据屏蔽、脱敏，数据一旦被删除，将导致您无法通过人脸识别方式进出冠寓门店）
4. 未成年人或无民事行为能力人、限制民事行为能力人本人无法作出上述授权的，须由其监护人确认并认可本授权协议；
5. 冠寓承诺，本次采集的人脸信息数据仅作为已授权用户使用人脸识别方式进出冠寓门店所用，不作他用。
